# BeyBladeLauncher

Wireless Beyblade Launcher using two ESP32-C3 communicating over BLE.

## Projects
- **BBLC** : BeyBlade Launcher Controller
- **BBLH** : BeyBlade Launcher Header

## Shared library
- **BleLink** : BLE communication layer (Central / Peripheral)

## Tooling
- PlatformIO
- ESP32-C3
- NimBLE
